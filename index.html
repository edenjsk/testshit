ChatGPT가 작성해준 GitHub Pages로 웹사이트 배포하기

GitHub Pages는 GitHub에서 정적 웹사이트를 무료로 호스팅할 수 있다! 방법은 유튜브에 널렀다!

1. 새로운 GitHub 리포지토리 생성

GitHub에 당연히 로그인 해야지. 새로운 리포지토리를 생성한다. 리포지토리 이름은 원하는 대로 설정, username.github.io 형식으로 리포지토리를 생성하면 기본 도메인이 된다.아무거나 써도 무관함.

2. index.html 파일 추가

리포지토리에 index.html 파일을 생성하고, 그 안에 웹사이트의 내용을 작성한다. 아무거나 해봐라. 당근 된다. 이 파일이 웹사이트의 홈 페이지로 작동한다.

3. GitHub Pages 설정

리포지토리 설정(Settings) 탭으로 이동한 다음, "Pages" 섹션을 찾습니다. "Source" 옵션에서 main 브랜치 또는 docs 폴더를 선택하고 "Save"를 클릭한다.

4. 웹사이트 접속

설정이 완료되면, 1분 정도 지나고 https://username.github.io에서 웹사이트 주소 하나 준다. 그럼 접속하기 클릭하면 바로 ㄱㄱ. 리포지토리 이름을 다르게 설정한 경우에는 https://username.github.io/repository-name 주소로 접속한다. 근데 그냥 QR code 써라.
